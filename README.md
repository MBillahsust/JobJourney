## Overview

**JobJourney** is a career preparation companion focused on engineering roles (SWE, Backend, SRE, Data; junior ‚Üí senior). It provides:

* A **fit analysis** between your r√©sum√© and a target JD
* A **gap-aware day-by-day plan** (default 7 days; ~10 hrs/week) with daily tasks and resources
* A **JD-aligned written exam** to benchmark your readiness
* **Tailored bullets, emails, and cover letters**
* An **application tracker** with reminders and calendar exports

> **Why day-by-day?** Instead of locking you into a fixed "14-day" label, the plan is rendered as **Day 1, Day 2, ‚Ä¶** with the option to extend, compress, or pause. This keeps momentum without forcing a specific duration.

## Workflow

A simple, explicit **work follow (workflow)** from discovery to offer. Each step has clear inputs/outputs and optional automations.

**Stages**

1. **Discover** ‚Äî Paste JD or pick from library ‚Üí JD summary & red flags
2. **Assess Fit** ‚Äî Match score, evidence view, gap list
3. **Plan** ‚Äî Generate **Day 1‚Ä¶Day N** plan (default N=7). Map gaps ‚Üí daily tasks
4. **Practice** ‚Äî Take exam; weak topics are injected back into Day N+ schedule
5. **Tailor** ‚Äî Create bullets, email, cover letter; all traceable to r√©sum√© facts
6. **Apply** ‚Äî Checklist + reminders (.ics export)
7. **Follow-up** ‚Äî Auto-draft follow-ups: recruiter nudge (2‚Äì3 days post-apply), thank-you (same day), status check (7‚Äì10 days)
8. **Track** ‚Äî Pipeline (Saved ‚Üí Applied ‚Üí Phone ‚Üí Onsite ‚Üí Offer ‚Üí Rejected), with notes/attachments

**Automations (optional)**

* Reminder presets per stage (e.g., **Send follow-up** 3 days after ‚ÄúApplied‚Äù)
* Calendar export for Day tasks & interviews
* Progress nudges when a day‚Äôs tasks are incomplete by 8pm local time

## How Scoring & Feedback Work

* **Auto-scored** items: MCQs, coding snippet tests, and SQL queries (with hidden test cases)
* **Rubric-based** scoring: short answers & system design mini (structure, correctness, trade-offs)
* **Section breakdowns:** e.g., SQL 6/10, System Design 7/10
* **Gap injection:** Low-scoring topics are **added to your gap list**; the **day-by-day plan** updates automatically
